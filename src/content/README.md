# Editor's quick reference

This folder is where all the words on the website live. Each file
controls one piece of the page. See the full guide at
[../../EDITING.md](../../EDITING.md) at the repo root.

| File                              | Controls                                |
| --------------------------------- | --------------------------------------- |
| `site.md`                         | Global settings (email, brand, nav)     |
| `home.md`                         | Hero section at the top of the page     |
| `services.md`                     | "What we do" section heading            |
| `services/*.md`                   | Individual service cards (one per file) |
| `about.md`                        | About section (bio, photo, sectors)     |
| `testimonials.md`                 | Testimonials section heading            |
| `testimonials/*.md`               | Individual quotes (one per file)        |
| `contact.md`                      | Contact section copy and buttons        |
